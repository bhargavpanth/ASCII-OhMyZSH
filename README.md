# ASCII-OhMyZSH

Add this file in any of your folder and obtain the PATH.

## Using bash?
1. Go to your terminal
2. ``` $ vi ~/.bashrc ```
3. ``` cat "Your path to the ASCII art file" ``` ex: ``` cat ~/Documents/ASCII/ascii.txt ```

## Using zsh?
1. Go to your terminal
2. ``` $ vi ~/.zshrc ```
3. ``` cat "Your path to the ASCII art file" ``` ex: ``` cat ~/Documents/ASCII/ascii.txt ```
